# シェルスクリプトで作成したスクリプト集です

### MySQL_ExportImportCopy
- MySQLのデータベースをコピーする

### PostgreSQL_Backup
- PostgreSQLのデータベースのDumpファイルの作成

### PostgreSQL_Import
- PostgreSQLのデータベースのDumpファイルからImportを行う

### ssh_connection
- 接続先設定ファイルを読み込み、その内容を画面に表示しどの環境に接続するかユーザーに対話し接続先を指定するツール

### PostgreSqlTool
- PostgreSQLのデータベースを操作する機能を提供するツール
